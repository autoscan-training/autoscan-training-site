ECU & CAN Training

CAN ойлголт + сүлжээний оношилгоо + ECU бүтэц (аюулгүй, зөв хэрэглээ).

Сурах зүйлс

  • CAN frame / ID / bitrate ойлголтын түвшинд
  • Сүлжээний wiring ба termination шалгах
  • Common network faults: open/short/termination issue
  • ECU бүтэц: flash/EEPROM ойлголт (суурь)
  • Лог/өгөгдөл дээр зөв тайлбар хийх

Анхааруулга

Энэ сургалт зөв, аюулгүй хэрэглээнд төвлөрнө.
Хамгаалалт тойрох, хууль зөрчих, буруу ашиглах аргачлал орохгүй.

Сурах үр дүн

  • CAN асуудлыг wiring + хэмжилтээр оношлох
  • Network fault-ыг системтэй тусгаарлах
  • ECU ойлголтоо нэг шат ахиулах
